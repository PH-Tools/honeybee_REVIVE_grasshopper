# honeybee_revive_grasshopper/ — the GH plugin package

The canvas-facing half: thin wrappers, compiled user objects, icons. Logic lives in `../honeybee_revive_rhino/`.

## Contents

- `src/` — thin GHPython component wrappers (one per component); instantiate the matching `GHCompo_*` and call `.run()`.
- `user_objects/` — compiled `.ghuser` binaries (regenerated in Grasshopper; do not hand-edit).
- `icons/` — component icons.

## Notes
- Wrapper filename, `ghenv.Component.Name`, and the `_component_info_.py` key must match. See `../context/ARCHITECTURE.md`.
